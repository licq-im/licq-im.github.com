Verify tarball

The tarballs are signed with Anders Olofsson's GPG key. To verify the tarball you've downloaded:

  1. Download the signature file matching the file you downloaded (1.7.1 or 1.7.0).
  2. Save the .sign file in the same directory as the tarball.
  3. Run gpg --verify <tarball>.sign (where <tarball> is e.g. licq-1.7.1.tar.bz2).
