Licq

We are currently moving Licq from our own Trac installation to GitHub so this page doesn't contain much info just yet.

Licq 1.5.0

The latest Licq release is 1.5.0 and can be downloaded from the download page.

Get the source

The source is available in our git repository.

Get help

If you have general questions post them to the licq-users mailing list. More development related questions and patches can be posted to the licq-dev mailing list.

Many of the developers also hang on the IRC channel #licq on chat.freenode.net. Drop by this IRC channel if you are interested in helping out, want some help, or just wanna hang out with the people developing your favourite instant messaging client.
